## Installation

Install printify-nodejs with npm

```bash
  npm install printify-nodejs
```
    
## Usage/Examples

```javascript
import { PrintifyClient } from 'printify-nodejs';

const client = new PrintifyClient("PRINTIFY_API_TOKEN");

const shops = await client.getShops();
```
